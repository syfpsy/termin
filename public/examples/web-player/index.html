Drop-in usage

Two tags. The web component loads the bundle, parses the .me source, and drives the engine.

<script type="module" src="/phosphor-player.js"></script>
<phosphor-player src="/scene.phosphor.json"></phosphor-player>

Inline bundle

Or embed the bundle directly — no extra fetch.

<phosphor-player>
  <script type="application/vnd.phosphor.bundle+json">
    { "schema": "phosphor.bundle.v1", "...": "..." }
  </script>
</phosphor-player>

Contract

Schema id phosphor.bundle.v1, MIME application/vnd.phosphor.bundle+json. The bundle keeps scene.source human-editable and ships scene.events for devices that can't parse the DSL.
